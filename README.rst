BEL Resources |build| |coverage|
================================
Utilities for BEL resource files.

Installation |pypi_version| |python_versions| |pypi_license|
------------------------------------------------------------
Install from `PyPI <https://pypi.python.org/pypi/bel-resources>`_ with:

.. code-block:: bash

   $ pip install bel-resources

or get the latest code from `GitHub <https://github.com/cthoyt/bel-resources>`_ with:

.. code-block:: bash

   $ git clone https://github.com/cthoyt/bel-resources.git
   $ cd bel-resources
   $ pip install -e .
